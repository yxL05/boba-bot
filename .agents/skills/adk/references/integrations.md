# Integrations

Integrations connect your agent to external platforms and services.

- Search available: `adk search <query>`
- Browse all: `adk list --available`
- Inspect details: `adk info <name>`
- Add to project: `adk add <name>@<version>`
- List installed: `adk list`

For detailed guidance — discovery, configuration types, lifecycle, common integrations — use the `adk-integrations` skill.

Install it with `npx skills add botpress/skills --skill adk-integrations`.
